HASHCODE
 HASHCODE is a linux sourcecode that can enable you encrypt or decrypt your text or message..
It was originally designed for the army so as to send unrecognizable message

INSTALLATION..
    $ apt update && apt upgrade
    $ apt install git
    $ apt install python
    $ apt install python2
    $ git clone https://github.com/Sup3r-Us3r/HashCode
    $ cd HashCode
    $ chmod +x *

RUN..
  • $ python3 hashcode-en.py
    At this step...it will display a banner and you will only have to follow the prompts now...Enjoy...
